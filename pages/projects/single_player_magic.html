Single Player Magic

version 1.0.0
Steam Workshop    Github    Supported version: 1.33.*
Single Player Magic is a mod that adds a cheat utility that is in-game which allows you to do various things in either Single or Multiplayer games that can be useful in various situations such as fixing something you do not agree with while trying to empire build, or simply just for fun casual empire building MP games, it is entirely up to the user, and this tool can be extremely helpful.
